proxy: proxy server settings (ip:port, with authentication)
description
Support proxy formats (only http): - 127.0.0.1:8000 - username:password@127.0.0.1:8000 - username:password:127.0.0.1:8000 - 127.0.0.1:8000@username:password - 127.0.0.1:8000:username:password
not implemented yet: support https/socks4/socks5
host ignore list: list of hosts to bypass proxy (direct connection)
description
In the input field, enter the host you want to bypass during proxying. You can remove a host from the list by clicking the "x" icon on the right. The list of hosts will update dynamically as you type, displaying the matched results.
not implemented yet: support wildcards (*.example.com, 192.168.*.*)
